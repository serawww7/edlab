🧮 Змішування добрив NPK: ціль → суміш → вартість → завантаження

🎯 Мета

За пару студенти підбирають суміш до 3 добрив під задану норму N–P₂O₅–K₂O, кг/га з урахуванням ціни і технічних обмежень. Результат: рецепт, відхилення, собівартість/га, план завантажень для усього поля та CSV для Sheets.

Математика: простий пошук прийнятного рішення (x≥0) із мінімальною сумою квадратичних відхилень по N,P,K з кроком для третього продукту.

🎯 Цілі внесення та параметри поля

Ціль (кг діючої речовини / га)

Поле та логістика

Керування

Порада: для старту скористайся «Приклад» (сечовина + DAP + MOP).

Формат: Назва | %N | %P₂O₅ | %K₂O | Ціна, грн/т

🪜 Хід роботи (45–70 хв)

  1. Заповни цілі N–P–K, площу, місткість бункера та таблицю добрив (або натисни «Приклад»).
  2. Відміть галочкою 2–3 продукти, якими дозволено користуватись, та натисни «Підібрати суміш».
  3. Перевір відхилення від цілі та собівартість. За потреби змінюй набір продуктів або ціль/допуск.
  4. Скопіюй CSV у Google Sheets: «Recipe» та «FieldLoads», побудуй діаграму ціни/га для різних рецептів.
