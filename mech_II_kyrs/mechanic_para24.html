📚 Завдання для студентів (вбудовано)

  1. Зроби **2 фото «до/після» очищення**. Для кожного збережи PNG (з heatmap) та CSV (метрики+клас).
  2. Побудуй **контрольну лінію** (Shift+клік ×2) і добийся |кут| < 2° для валів/шківів. Додай запис у журнал.
  3. Навчи k-NN: додай **по 3–5 прикладів** на клас (oil/rust/…). Повтори класифікацію і порівняй зміни. Експортуй JSON із прикладами.
  4. Заповни **ISO-чек-лист** (галочки), сформуй **TXT-звіт** та **чек 58 мм**.
